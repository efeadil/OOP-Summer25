**📌Data Types**
print(type(10))  # int
print(type("Hello"))  # str
print(type(3.14))  # float

print(type((1, 2, 3)))  # tuple
print(type({1, 2, 3}))  # set
print(type({"key": "value"}))  # dict

**📌 Conditional Statements**
x = 10
if x > 5:

elif x == 5:

**📌 Loops (For & While)**
    # For loop

# While loop
count = 0
while count < 5:
    print(count)
    count += 1
